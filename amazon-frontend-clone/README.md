# 🛒 Amazon Clone 🛍️  

A clone of the Amazon website built with **HTML, CSS & JavaScript**. It includes features such as viewing products, adding to the cart, checkout process, and more.   

## 🛠️ Tech Stack  
🔹 **HTML** – Structure  
🔹 **CSS** – Styling  
🔹 **JavaScript** – Interactivity & Logic
